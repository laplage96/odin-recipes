# Odin Recipes

## Description
This project is part of the Foundations course at The Odin Project. It is a basic recipe website consisting of a main index page that links to various recipe pages.

## Skills Demonstrated
By completing this project, I have demonstrated the following skills:
* **HTML Structure**: Using proper boilerplate, headings, and body tags.
* **Text Formatting**: Implementing paragraphs, bolding, and italics for readability.
* **Lists**: Utilizing ordered and unordered lists for ingredients and instructions.
* **Links**: Connecting multiple HTML documents using relative file paths.
* **Git Workflow**: Managing a local repository and pushing changes to a remote repository on GitHub.

## Future Plans
Once I dive deeper into the course, I plan to return to this project to apply **CSS** for layout and styling.
